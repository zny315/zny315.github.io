# Neyan Zhao Academic Homepage

This is a dependency-free static website. Open `index.html` directly to preview it.

## Publish with GitHub Pages

1. Create a public GitHub repository and upload `index.html`, `styles.css`, and the `assets` folder.
2. Open the repository's **Settings > Pages**.
3. Under **Build and deployment**, select **Deploy from a branch**, then choose the `main` branch and `/ (root)` folder.
4. GitHub will provide a public address after the first deployment finishes.

## Publish with Netlify

Drag the entire `zhao-homepage` folder into Netlify Drop. Netlify will generate a public address without a build command.

Before publishing, review the public email address and update the copyright year or publication list when needed.
